# QA Automation
Projet d’automatisation pour examen QA – Panier → Paiement

Ce projet démontre la capacité à automatiser des tests UI et API pour un flux e‑commerce simple (ajout au panier, mise à jour de quantité, validation d’achat).  

Il inclut :

- Tests UI Cypress (mockés via `cy.intercept()`)
- Tests API Cypress (GET/POST simulés)
- Exemples de résultats d’exécution (logs, screenshots, rapports)
